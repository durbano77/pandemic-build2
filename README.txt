In order to compile the program in visual studio (2015) you must:

1. Go into the "Project" tab near the top-left of the screen.
2. Go into [Project Name] Properties.
3. Go into "Configuration Properties" -> "General" (which should be the default screen it opens to when opening project properties).
4. Under "Use of MFC", select the "Use MFC in a shared DLL".
5. Press "Apply" to confirm the changes.
